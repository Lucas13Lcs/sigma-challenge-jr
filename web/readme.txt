Projeto - Democraticamente 
Eleições 2018 - Sigma 

Objetivos:
 *Criar uma aplicação Web utilizando a API do TSE como fonte de dados, para listragem e apresentação de candidatos as eleições 2018, com os seguinte cargos: Presidente, Governador, Senador, Deputado Federal e Deputado Estadual

Requisitos:
  * Página para listagem de candidatos.
  * Página para visualização dos detalhes do candidato.

Procedimentos:
Etapa 1 - Listagem de candidatos:
   * Elaboração do layout da tela.
   * Criação de layout responsivo.
   * Estilização.
   * Testes em diferentes dispositivos.
   * Codificação das regras de negócio.
   * Testes

Etapa 2 - Tela de detalhes dos candidatos
   * Elaboração do layout da tela.
   * Criação de layout responsivo.
   * Estilização.
   * Testes em diferentes dispositivos.
   * Codificação das regras de negócio.
   * Testes.


Funcionalidades do Sistema:

1 - Listagem de Candidatos.

   1.1 Slides expliativos sobre os cargos em disputa.
   1.2 Busca de candidatos baseado no cargo.
	1.2.1 Com filtro por cargo, listando todos os candidatos na disputa.	
	1.2.2 Com filtro por Cargo e Nome do candidato.
	1.2.3 Com filtro por Cargo e Número do candidato.
 	1.2.4 Com filtro por Cargo, Nome do Candidato e Número do Candidato.
   1.3 Os resultado filtrados são exibidos em uma lista, com Nome do candidato, número e sigla do partido. 	
   1.4 Ao clicar em um item da lista, direcina-se para uma página com detalhes do candidato.

2 - Detalhes dos Canditados.

   2.1 Layout com aba: Dados Pessoais | Cargo | Partido | Vice Candidato, para os cargos de Presidente, Governador e Senador.
   2.1 Layout com aba: Dados Pessoais | Cargo | Partido,  para os cargos de Deputado Federal e Deputado Estadual10:22 01/10/2018
   2.2 Foto do candidato em destaque nas abas: Dados Pessoais | Cargo | Partido.
   2.3 Foto do vice candidato em destaque na aba Vice Candidato.
   2.4 Botão para retornar a página de Listagem de candidatos.
